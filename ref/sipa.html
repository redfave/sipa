class sipa.flatpages.Article(extension, parent, article_id)[source]¶

Bases: sipa.flatpages.Node

The Article class

An article provides the possibility to access multiple versions of a Page. In this case, localized_pages is a dict where a locale string points to a Page. The latter represents the actual markdown file located in the repository.

After the initialization, which consists of adding pages with add_page(), internal methods access only the page with the correct locale, which is proxied by localized_page.

Besides that, __getattr__() comfortably passes queries to the localized_page.meta dict.

sipa.mail module¶

Functions concerned with mail composition and transmission

This module ultimately provides functions to be used by a blueprint in order to compose and send a mail, for instance send_contact_mail().

On the layer below, some intermediate functions are introduced which are needed to compose and send the mails. The core is send_complex_mail(), which calls send_mail() prepending optional information to the title and body
