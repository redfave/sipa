sipa¶

The sipa package contains everything – things that are not grouped in subpackages are mainly frontend-related.

.blueprints¶

The sipa.blueprints package contains the actual endpoint definitions. For an overview of flask blueprints, refer to the flask documentation.

.flatpages¶

The sipa.flatpages module provides the CategorizedFlatPages flask extension, which is based on Flask-FlatPages.
